# Suneditor field for [MoonShine Laravel admin panel](https://moonshine-laravel.com)

### Requirements

- MoonShine v2.0+

## Installation
```shell
composer require tmoiseenko/moonshine-suneditor
```

## Usage

```php
use Tmoiseenko\MoonshineSuneditor\Fields\Suneditor;

Suneditor::make('Label')
```
